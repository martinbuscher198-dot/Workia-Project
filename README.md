# Workia-Project
Es un proyecto en el cual clientes pueden contratar y reservar servicios 


Como usar la pagina

Como esta pagina no esta subida a ningun dominio necesitaremos descargar xampp, cuando abramos xampp en Apache y MySQL le damos start,
posterior a eso le damos en "admin" en MYSQL, vamos a importar, vamos abajo del todo le damos al boton importar e importamos el archivo .sql, y con eso la paginaria estaria funcional!
